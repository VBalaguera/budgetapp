# Welcome to Carrington

This is a simple budget and note taking app that uses:

- React.
- Chart.js
- i18next.js
- bootstrap v5.

Using localstorage, all users can safely save their content in the browser.

### Installing dependencies:

```sh
npm install
```

### Starting app:

```sh
npm run start

```

Enjoy.
